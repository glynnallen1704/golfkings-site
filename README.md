# golfkings-site

Static marketing site for the GolfKings iOS/Android app. Lives at
[https://glynnallen1704.github.io/golfkings-site/](https://glynnallen1704.github.io/golfkings-site/).

## What's here

- `index.html` — marketing landing page
- `privacy.html` — privacy policy (required by App Store Connect)
- `support.html` — support FAQ + contact (required by App Store Connect)
- `styles.css` — shared design system

## Stack

Plain HTML/CSS. No build step. No framework. Hosted free on GitHub Pages.

## Editing

1. Edit the relevant file
2. Commit + push to `main`
3. GitHub Pages republishes within ~30 seconds

## Future

If we register `golfkings.app`, the same `gh-pages` deploy can attach the
custom domain via the repo Settings → Pages → Custom domain field.
